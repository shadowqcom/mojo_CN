## Hello Mojo

在你喜欢的 IDE 或编辑器中创建 `analyzer.mojo` 文件。

在 `analyzer.mojo` 中添加：

```mojo
def main():
    print("温度分析器")
```

运行它：

```bash
mojo analyzer.mojo
```

要点：

- 如果看到"温度分析器"，说明你的设置正常工作。
- 所有 Mojo 可执行程序都使用 `main()` 作为入口点。

## 变量和数据

更新你的文件，添加温度数据：

```mojo
def main():
    print("温度分析器")

    # 方括号告诉 Mojo 在编译时 `List` 的类型
    var temps: List[Float64] = [20.5, 22.3, 19.8, 25.1]

    print("记录了", len(temps), "个温度数据")
```

## 循环

打印每个温度。在 `main()` 中的 print 语句后添加：

```mojo
def main():
    # ... 已有代码 ...

    for index in range(len(temps)):  # 范围是 [0, len(temps))
        print("  第{}天: {}°C".format(index + 1, temps[index]))
```

要点：

- 天数偏移 1 是因为范围使用从零开始的索引。

## 函数

在 `main()` 上方添加此函数来计算平均温度：

```mojo
def calculate_average(temps: List[Float64]) -> Float64:
    var total: Float64 = 0.0
    for index in range(len(temps)):
        total += temps[index]
    return total / Float64(len(temps))

def main():
    # ... 已有代码 ...
```

在 `main()` 末尾添加调用函数的代码：

```mojo
    var avg = calculate_average(temps)
    print("平均值: {}°C".format(avg))
```

## 条件语句

对平均温度进行分类。在 `main()` 末尾添加：

```mojo
    if avg > 25.0:
        print("状态: 炎热的一周")
    elif avg > 20.0:
        print("状态: 舒适的一周")
    else:
        print("状态: 凉爽的一周")
```

## 抛出错误

通过更新 `calculate_average` 来处理空数据。现在函数可以抛出错误了。

```mojo
def calculate_average(temps: List[Float64]) raises -> Float64:
    if len(temps) == 0:  # 空的温度列表
        raise Error("没有温度数据")

    var total: Float64 = 0.0
    for index in range(len(temps)):
        total += temps[index]
    return total / Float64(len(temps))
```

变化：

- 在返回箭头前添加 `raises`。
- 添加空列表检查。
- 如果为空则抛出 `Error`。

## 处理错误

在 `main()` 中，用 `try-except` 包装你的代码以处理错误：

```mojo
    try:
        var avg = calculate_average(temps)
        print("平均值: {}°C".format(avg))

        if avg > 25.0:
            print("状态: 炎热的一周")
        elif avg > 20.0:
            print("状态: 舒适的一周")
        else:
            print("状态: 凉爽的一周")
    except e:
        print("错误:", e)
```

要测试错误，将 `temps` 替换为 `List[Float64]()`。
确认你的应用程序输出"没有温度数据"错误。

## 完整代码

你的完整 `analyzer.mojo`：

```mojo
from std.python import Python, PythonObject

def calculate_average(temps: List[Float64]) raises -> Float64:
    if len(temps) == 0:
        raise Error("没有温度数据")

    var total: Float64 = 0.0
    for index in range(len(temps)):
        total += temps[index]
    return total / Float64(len(temps))

def main():
    print("温度分析器")
    var temps: List[Float64] = [20.5, 22.3, 19.8, 25.1]
    print("记录了", len(temps), "个温度数据")

    for index in range(len(temps)):
        print("  第{}天: {}°C".format(index + 1, temps[index]))

    try:
        var avg = calculate_average(temps)
        print("平均值: {}°C".format(avg))

        if avg > 25.0:
            print("状态: 炎热的一周")
        elif avg > 20.0:
            print("状态: 舒适的一周")
        else:
            print("状态: 凉爽的一周")

        var np = Python.import_module("numpy")
        var pytemps: PythonObject = [20.5, 22.3, 19.8, 25.1]
        print("温度标准差:", np.std(pytemps))
        _ = pytemps^ # 允许 Python 对象释放
    except e:
        print("错误:", e)
```
