# len

Provides the `len()` function and its associated traits.

These are Mojo built-ins, so you don't need to import them.

## Traits

### Sized

`trait Sized`

**Methods:**

- **__len__**: `def __len__(self) -&gt; Int`

### SizedRaising

`trait SizedRaising`

**Methods:**

- **__len__**: `def __len__(self) raises -&gt; Int`
